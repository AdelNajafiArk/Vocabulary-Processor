A Pure PHP Vocabulary Learning Platform with Interactive Games

Vocabulary Master Pro is a feature-rich, self-hosted web application built entirely with pure PHP (no frameworks) for English vocabulary enhancement. It combines comprehensive learning tools with engaging games to make vocabulary acquisition effective and enjoyable.
Key Features
Learning Core

    WordNet Integration: Synonyms, definitions, and examples

    Daily Delivery: 10 random words each day

    Smart Import: PDF/Excel/CSV support for academic vocabulary

    Progress Tracking: Personal statistics and learning history

    Audio Tools: Text-to-speech pronunciation and automatic phonetics

Interactive Games (5 Unique Types)

    Word Matching: Match words with definitions

    Speed Challenge: Type definitions against the clock

    Word Puzzle: Complete missing letters in words

    Vocabulary Battle: Compete in knowledge challenges

    Word Categories: Sort words into semantic groups

Assessment System

    Vocabulary tests with multiple-choice questions

    Targeted review of incorrect answers

    Performance analytics and progress visualization

    Adaptive difficulty based on user level
